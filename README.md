### 🚀 Sobre mí

🔧 Cuento con más de 3 años de experiencia desarrollando servicios backend en entornos productivos, trabajando con **Node.js**, **TypeScript** y múltiples motores de base de datos.

🧠 Me enfoco en la resolución de problemas reales, optimización de consultas SQL y mejora de performance en sistemas existentes.

📊 Tengo experiencia trabajando con **Oracle, PostgreSQL, SQL Server y MongoDB**, incluyendo debugging de errores, joins complejos y análisis de impacto.

🔗 Desarrollo integraciones entre sistemas mediante **APIs REST**, mensajería (**RabbitMQ**) y procesamiento en tiempo real.

⚙️ Automatizo procesos mediante **cron jobs** y herramientas de backend para mejorar eficiencia operativa.

🧪 Trabajo con testing y calidad utilizando **Jest** y **Supertest**.

🐳 Actualmente continúo fortaleciendo conocimientos en **Docker**, despliegue y entornos productivos.
